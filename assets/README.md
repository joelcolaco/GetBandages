# Asset Pack

This folder contains an original SVG starter asset pack for Get Bandages.

## Included
- Backgrounds: day and night sky scenes
- Terrain tiles: ground and platform textures
- Hazard: spike strip
- Player sprites: idle, run (2-frame), jump, dead
- Enemy sprites: slime (2-frame)
- Prop: goal flag
- UI icons: bandage and heart

## Notes
- All files are SVG for easy editing and crisp scaling.
- Use `assets/manifest.json` as the source of truth for loading paths.
